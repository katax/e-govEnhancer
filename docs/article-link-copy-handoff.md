# 条文リンクコピー 引き継ぎメモ

## 概要

e-Gov 法令画面で `a` キーを押すと「条文リンクコピー」モードに入り、`↑↓` で選択対象を切り替え、コピー系ショートカットで選択中要素の内容をクリップボードへコピーする機能を実装中。

対象は以下。

- 条
- 項
- 号

## 現在のショートカット仕様

- `a`
  - 条文リンクコピーモードを開く
- `↑` / `↓`
  - 選択対象の移動
- `Enter`
  - URL のみコピー
- `Ctrl+Enter`
  - 選択中の条文本文 + 改行 + URL をコピー
- `Shift+Enter`
  - `法令名 + 条文番号` + 改行 + URL をコピー
- `Esc`
  - モード終了

コピー成功時はモードを閉じる仕様にしてある。

### 追加済み機能

- `content.js`
  - `a` キーで `showArticleLinkCopyDialog()` を起動

  - `Enter / Ctrl+Enter / Shift+Enter` の3種類のコピー内容生成
  - 選択対象のページ上赤枠表示

## いま困っている課題

### 最重要課題

`Ctrl+Enter` で「条全体」をコピーしたときの改行復元がまだ正しくない。

特に、条全体に複数項がある場合に、本来は

```text
（見出し）
第三条の二　……
２　……
３　……
https://...
```

であるべきなのに、実際には

```text
（見出し）
第三条の二　……。２　……。３　……。
https://...
```

のように `。２` や `。３` の前で改行されない。

### 追加で起きた関連課題

`Ctrl+Enter` の本文コピーまわりで、次の両立が必要。

1. 項番号や条番号の直後では改行しない  
   例:

```text
第三条の四　消費者……
```

```text
２　労働契約の存否……
```

2. ただし、条全体コピー時は項の境目では改行する  
   例:

```text
第三条の二　……
２　……
３　……
```

3. 条見出しがある場合は見出しと本文の間で改行する  
   例:

```text
（簡易裁判所の裁量移送）
第十八条　……
```

## 実現したい最終仕様

### 1. 選択モード

- `a` でモード開始
- `↑↓` で条・項・号を選択
- 選択中要素はページ上で赤枠表示
- コピー成功時はモードを閉じる

### 2. ダイアログ表示

ダイアログには以下のみ表示。

```text
Enterでコピーされる内容
[プレビュー]
Shift+Enterでコピーされる内容
[プレビュー]
Ctrl+Enterでコピーされる内容
[プレビュー]
---
操作ガイド
↑↓ で選択｜Escでキャンセル
```

補足:

- `↑` `↓` `Esc` だけ `kbd` 風囲い
- 操作ガイドはプレビュー欄と同じ背景にしない

### 3. コピー内容

#### `Enter`

```text
URLのみ
```

#### `Shift+Enter`

```text
法令名 条文番号
URL
```

例:

```text
民事訴訟法 第三条の二
https://...
```

補足:

- 法令名末尾の `（平成八年法律第百九号）` のような法令番号括弧は除去する

#### `Ctrl+Enter`

```text
条文本文
URL
```

補足:

- 項番号や条番号の直後では改行しない
- 条全体コピーでは見出しと各項の区切り改行は維持する
- つまり HTML 構造に応じて行を組み立てる必要がある
