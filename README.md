# e-Gov法令検索 Enhancer

この拡張機能は、`https://laws.e-gov.go.jp/` 向けの Chrome 拡張機能です。

`Chrome` や `Microsoft Edge` では、ストアに公開していない拡張機能でも「パッケージ化されていない拡張機能」として読み込めます。ここでは、この GitHub リポジトリから ZIP をダウンロードしてインストールする手順を説明します。

## インストール手順

1. 右上付近の `Code` ボタンを押して、`Download ZIP` を選びます。

2. ダウンロードしたZIPファイルを展開します。（ZIPファイルを右クリックして「すべて展開」を選ぶ）

3. ブラウザで拡張機能管理画面を開きます。
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`

4. 右上の「デベロッパー モード」を `ON` にします。

5. 「パッケージ化されていない拡張機能を読み込む」をクリックします。

6. ZIPを展開してできたフォルダのうち、`manifest.json` が入っているフォルダを選択します。（変更していなければe-pubEnhancer-mainというフォルダ名のフォルダのはずです）

7. 一覧に `e-Gov法令検索 Enhancer` が表示されたらインストール完了です。

## 更新方法

GitHubから新しいZIPをダウンロードして展開し直したあと、拡張機能管理画面でこの拡張機能の `再読み込み` を押してください。

## オプション画面の開き方

拡張機能管理画面のこの拡張機能カードから `詳細` を開き、`拡張機能のオプション` を開いてください。

## 注意

- ブラウザをまたいで使う場合は、それぞれのブラウザで読み込みが必要です。
