ショートカットキー設定

現在のショートカットキー 読み込み中...

ショートカットキーは Chromeの拡張機能管理ページ から変更できます。
デフォルト設定：Ctrl+Shift+E / F / H（macOS：⌘+Shift+E / F / H）

  • E：法令検索画面
  • F：お気に入り画面
  • H：開いた法令の履歴画面

画面を開いた後は、左右カーソルキーで各モードを移動できます。

📖

使い方ガイド

🌐

どこからでも法令を検索

Chrome上の任意のページでショートカットキーを押すと、ポップアップが開きます。 法令名やキーワードを入力して検索すると、新しいタブでe-Gov法令検索の結果が表示されます。

⭐

お気に入り・開いた法令履歴

検索結果の ☆ボタン または履歴パネルの Shift+Enter でお気に入りに追加できます。 ポップアップで ◀（左矢印キー）で「開いた法令」一覧、▶（右矢印キー）でお気に入りを表示します。 お気に入りはフォルダで整理でき、ドラッグ＆ドロップで移動できます。

🗂️

法令ページのキーボードショートカット

e-Govの法令ページを開いているとき、入力欄にフォーカスがない状態でキーを押すと機能します。

? で操作ガイドを表示できます。

キー機能
0〜9 条文ジャンプ — 条番号を入力して Enter で該当条文へ移動。. で項・号を指定（例: 3.2.1 → 第3条第2項第1号）
h / l 条文ジャンプ履歴の前後に移動
n / p 次の条文 / 前の条文を画面上端に表示
d / u 画面を80%下スクロール / 上スクロール
s ページ内検索 — キーワードをハイライト。Enter で次の一致箇所、Ctrl+Enter で現在位置から検索
r 現在表示中の法令名でポップアップ検索（別タブで開く）
f 現在の法令をお気に入りに追加 / 解除
a 条文リンクコピー ー 条・項・号を ↑↓ で選び、Enter でURL、Ctrl+Enter で条項＋URL、Shift+Enter で法令名＋条文番号＋URLをコピー
b ピン状態の常時表示を切り替え
i o j k m 対応色のピンを設定 / 同じ条項なら解除 / 別条項ならそのピンへ移動
Shift+I / O / J / K / M 対応色のピンを強制解除
c 条文番号の漢数字 ↔ アラビア数字を切り替え（号タイトルは丸数字 ①②③）
g 本文中の括弧書きを薄く表示 / 元に戻す
Shift+G 本文中の括弧書きをさらに薄く表示 / 元に戻す
Shift+H ページ内のカタカナをひらがなに変換（一回のみ）
Alt+O このオプション画面を開く
Alt+P 上記すべてのキーボードショートカットを一時的に有効 / 無効に切り替え（右下ボタンの色で確認：青=有効、灰=無効）
右下 ⌨️ ボタンクリック ショートカットの有効 / 無効を切り替え（ホバーでショートカット一覧を表示）
Esc ダイアログを閉じる
